  <name>Task 2: Apply RLS policies for tenant isolation across all rfp_* tables</name>
  <files>supabase/migrations/20260509_rfp_rls_policies.sql</files>
  <action>
Create `supabase/migrations/20260509_rfp_rls_policies.sql` and apply via supabase MCP `apply_migration` (project `hgxxxmtfmvguotkowxbu`, name `rfp_rls_policies_v1`).

The policy model:
- `rfp_user_orgs` is the membership source of truth.
- A user can read/write a row in any tenant-scoped table iff that row's `org_id` is in `(SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid())`.
- Role-based gates (writer vs viewer) apply on top — viewer can SELECT but not INSERT/UPDATE/DELETE; writer can do all four; reviewer can read + insert into rfp_compliance_checks; owner can do everything plus delete the org itself.

For each table below, `ENABLE ROW LEVEL SECURITY`, then apply the listed policies. Use named policies so they're traceable.

**Tables and their policies:**

1. `rfp_orgs` — direct membership check (no org_id; org_id IS the row id):
   - `rfp_orgs_select`: `FOR SELECT USING (id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid()))`
   - `rfp_orgs_insert`: `FOR INSERT WITH CHECK (true)` (any authenticated user can create an org; the API immediately inserts a rfp_user_orgs(owner) row in the same transaction)
   - `rfp_orgs_update`: `FOR UPDATE USING (id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid() AND role IN ('owner')))`
   - `rfp_orgs_delete`: `FOR DELETE USING (id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid() AND role = 'owner'))`

2. `rfp_users` — self-only:
   - `rfp_users_select_own`: `FOR SELECT USING (id = auth.uid())`
   - `rfp_users_upsert_own`: `FOR ALL USING (id = auth.uid()) WITH CHECK (id = auth.uid())`

3. `rfp_user_orgs` — own memberships + org owners can manage:
   - `rfp_user_orgs_select`: `FOR SELECT USING (user_id = auth.uid() OR org_id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid() AND role = 'owner'))`
   - `rfp_user_orgs_insert`: `FOR INSERT WITH CHECK (user_id = auth.uid() OR org_id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid() AND role = 'owner'))` (lets a user accept their own invite, lets an owner add someone)
   - `rfp_user_orgs_delete`: `FOR DELETE USING (user_id = auth.uid() OR org_id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid() AND role = 'owner'))`

4. **For each of `rfp_capture_profiles`, `rfp_vault_artifacts`, `rfp_proposals`, `rfp_compliance_checks`, `rfp_agent_sessions`, `rfp_opp_matches`** (the tenant-scoped tables — they all have an `org_id` column): apply this 4-policy template:
   - `{table}_select`: `FOR SELECT USING (org_id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid()))`
   - `{table}_insert`: `FOR INSERT WITH CHECK (org_id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid() AND role IN ('owner','writer','reviewer')))`
   - `{table}_update`: `FOR UPDATE USING (org_id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid() AND role IN ('owner','writer')))`
   - `{table}_delete`: `FOR DELETE USING (org_id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid() AND role = 'owner'))`

5. `rfp_proposal_sections` — section is gated by its parent proposal's org. Use a subquery rather than denormalize:
   - `rfp_proposal_sections_select`: `FOR SELECT USING (proposal_id IN (SELECT id FROM rfp_proposals WHERE org_id IN (SELECT org_id FROM rfp_user_orgs WHERE user_id = auth.uid())))`
   - Mirror INSERT/UPDATE/DELETE with role gates as above, joining through rfp_proposals.

6. `rfp_opportunities` — global read for any authenticated user (opportunities are not tenant-scoped); only service role writes (cron worker uses admin client):
   - `rfp_opportunities_select`: `FOR SELECT USING (auth.uid() IS NOT NULL)`
   - No INSERT/UPDATE/DELETE policy → only `service_role` can write (Supabase admin client bypasses RLS).

After applying, run `get_advisors` (type=`security`) via supabase MCP. Expected: zero `rls_disabled_in_public` errors on any `rfp_*` table. If any flag, fix the migration before moving on.

Test structure:

```typescript
import { describe, it, expect, beforeAll, afterAll } from 'vitest';
import { createClient } from '@supabase/supabase-js';

const URL = process.env.NEXT_PUBLIC_SUPABASE_URL!;
const ANON = process.env.NEXT_PUBLIC_SUPABASE_ANON_KEY!;
const SERVICE = process.env.SUPABASE_SERVICE_ROLE_KEY!;

const admin = createClient(URL, SERVICE, { auth: { persistSession: false } });

let userA: { id: string; access_token: string };
let userB: { id: string; access_token: string };
let orgA: string;
let orgB: string;

beforeAll(async () => {
  // Create two ephemeral users via admin
  const a = await admin.auth.admin.createUser({ email: `rls-test-a-${Date.now()}@test.local`, password: 'test-password-12345', email_confirm: true });
  const b = await admin.auth.admin.createUser({ email: `rls-test-b-${Date.now()}@test.local`, password: 'test-password-12345', email_confirm: true });

  // Sign each in to get a JWT
  const sA = await createClient(URL, ANON).auth.signInWithPassword({ email: a.data.user!.email!, password: 'test-password-12345' });
  const sB = await createClient(URL, ANON).auth.signInWithPassword({ email: b.data.user!.email!, password: 'test-password-12345' });

  userA = { id: a.data.user!.id, access_token: sA.data.session!.access_token };
  userB = { id: b.data.user!.id, access_token: sB.data.session!.access_token };

  // Admin creates two orgs and memberships
  const oA = await admin.from('rfp_orgs').insert({ name: 'Org A', type: 'nonprofit' }).select().single();
  const oB = await admin.from('rfp_orgs').insert({ name: 'Org B', type: 'forprofit' }).select().single();
  orgA = oA.data!.id;
  orgB = oB.data!.id;
  await admin.from('rfp_user_orgs').insert([
    { user_id: userA.id, org_id: orgA, role: 'owner' },
    { user_id: userB.id, org_id: orgB, role: 'owner' },
  ]);

  // Seed a vault artifact in each org
  await admin.from('rfp_vault_artifacts').insert([
    { org_id: orgA, type: 'past_proposal', title: 'A-secret', body: 'A only' },
    { org_id: orgB, type: 'past_proposal', title: 'B-secret', body: 'B only' },
  ]);
});

afterAll(async () => {
  // Teardown: delete orgs (cascades) and users
  await admin.from('rfp_orgs').delete().in('id', [orgA, orgB]);
  await admin.auth.admin.deleteUser(userA.id);
  await admin.auth.admin.deleteUser(userB.id);
});

function clientFor(token: string) {
  return createClient(URL, ANON, { global: { headers: { Authorization: `Bearer ${token}` } } });
}

describe('RFP tenant isolation', () => {
  it('User A cannot read Org B vault artifacts', async () => {
    const c = clientFor(userA.access_token);
    const { data } = await c.from('rfp_vault_artifacts').select('*').eq('org_id', orgB);
    expect(data).toEqual([]);
  });

  it('User A reads only Org A vault artifacts', async () => {
    const c = clientFor(userA.access_token);
    const { data } = await c.from('rfp_vault_artifacts').select('*');
    expect(data?.every(r => r.org_id === orgA)).toBe(true);
    expect(data?.length).toBeGreaterThan(0);
  });

  it('User A cannot insert into Org B', async () => {
    const c = clientFor(userA.access_token);
    const { error } = await c.from('rfp_vault_artifacts').insert({ org_id: orgB, type: 'partner_letter', title: 'sneak', body: 'no' });
    expect(error).not.toBeNull();
  });

  it('User A cannot update Org B rows', async () => {
    const c = clientFor(userA.access_token);
    const { error, data } = await c.from('rfp_vault_artifacts').update({ title: 'pwned' }).eq('org_id', orgB).select();
    // Either error OR empty result (RLS hides the row from UPDATE)
    expect(error !== null || (data?.length ?? 0) === 0).toBe(true);
  });

  it('User B sees their own org in rfp_orgs but not Org A', async () => {
    const c = clientFor(userB.access_token);
    const { data } = await c.from('rfp_orgs').select('id, name');
    expect(data?.find(o => o.id === orgB)).toBeTruthy();
    expect(data?.find(o => o.id === orgA)).toBeUndefined();
  });

  it('rfp_opportunities is globally readable for both users', async () => {
    // Seed via admin
    const opp = await admin.from('rfp_opportunities').insert({ source: 'sam_gov', source_id: `test-${Date.now()}`, title: 'Public opp', deadline: new Date(Date.now() + 86400000).toISOString() }).select().single();
    const cA = clientFor(userA.access_token);
    const cB = clientFor(userB.access_token);
    const rA = await cA.from('rfp_opportunities').select('*').eq('id', opp.data!.id).single();
    const rB = await cB.from('rfp_opportunities').select('*').eq('id', opp.data!.id).single();
    expect(rA.data?.id).toBe(opp.data!.id);
    expect(rB.data?.id).toBe(opp.data!.id);
    await admin.from('rfp_opportunities').delete().eq('id', opp.data!.id);
  });
});
```
