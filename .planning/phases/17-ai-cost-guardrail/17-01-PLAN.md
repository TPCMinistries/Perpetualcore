Create `lib/rfp/ai/guardrail.ts` (new directory lib/rfp/ai/). TypeScript strict, no `any`. Export the following:

1. `class BudgetExceededError extends Error` ŌĆö with readonly `code = "BUDGET_EXCEEDED"`, `orgId: string`, `spentUsd: number`, `limitUsd: number`. Message: `AI budget exceeded: $<spent> spent of $<limit> limit`.

2. `interface LLMCallMeta` ŌĆö `{ agent: string; proposalId?: string; sessionId?: string; model: string; tokensIn: number; tokensOut: number; costUsd: number }`. This is what wrapped fns return.

3. `const RFP_MODEL_RATES` ŌĆö canonical, easy-to-edit map keyed by model id, each `{ inputPerM: number; outputPerM: number }` (USD per 1M tokens). Seed with current rates from research: gpt-4o {2.50, 10.00}, text-embedding-3-large {0.13, 0} (embeddings have no output), claude-sonnet-4-5 {3.00, 15.00}, claude-haiku-4-5 {0.25, 1.25}. Add a one-line comment at the top noting "rates as of 2026-06; update here only." Also export a helper `computeCostUsd(model: string, tokensIn: number, tokensOut: number): number` that looks up the rate (throws or returns a safe 0-with-warn if model unknown ŌĆö choose warn-and-estimate-with-0 so an unknown model never blocks a call, but log it). The call sites already compute their own cost; this map is the canonical reference and is used by the cron/summary path which has no local pricing.

4. `async function guardedLLMCall<T extends LLMCallMeta>(orgId: string, fn: () => Promise<T>): Promise<T>`:
   - Call `checkBudget(orgId)` first (throws BudgetExceededError if over). If checkBudget throws BudgetExceededError, let it propagate ŌĆö fn() must NEVER be called.
   - Then `const result = await fn();`
   - Then `await recordCost(orgId, result).catch((err) => console.error("[guardrail] cost record failed (non-fatal):", err));`
   - Return result.

5. `async function checkBudget(orgId: string): Promise<void>` (internal, not exported unless useful for tests ŌĆö export it to allow unit testing):
   - `const admin = createAdminClient();` from `@/lib/supabase/server`.
   - Read `monthly_ai_budget_usd` from rfp_entitlements where org_id = orgId (.maybeSingle()).
   - Explicit null guard: `if (limitUsd === null || limitUsd === undefined) return;` (NULL = unlimited). Do NOT let NULL coerce to 0.
   - Compute calendar-month start in UTC: `new Date(Date.UTC(now.getUTCFullYear(), now.getUTCMonth(), 1, 0,0,0,0))` and use `.toISOString()`. (Calendar month, NOT rolling 30 days.)
   - Sum this month's cost_usd from rfp_agent_sessions where org_id = orgId AND created_at >= monthStart AND cost_usd is not null. Numbers may come back as string|number ŌĆö coerce with a `toNum` helper (Number(x) || 0).
   - If `spentUsd >= limitUsd` throw `new BudgetExceededError(orgId, spentUsd, limitUsd)`.
   - If the Supabase read itself ERRORS (not just empty): do NOT fail-open. Re-throw as a normal Error so the route returns 500 (DB error, user can retry) ŌĆö do NOT swallow into a budget pass.

6. `async function recordCost(orgId: string, meta: LLMCallMeta): Promise<void>` (internal):
   - `const admin = createAdminClient();`
   - Insert one rfp_agent_sessions row: `{ org_id: orgId, proposal_id: meta.proposalId ?? null, agent: meta.agent, session_id: meta.sessionId ?? null, model: meta.model, tokens_in: meta.tokensIn, tokens_out: meta.tokensOut, cost_usd: meta.costUsd }`.

7. `function budgetExceededResponse(err: BudgetExceededError): NextResponse` ŌĆö import NextResponse from "next/server". Returns status 402 with body `{ error: "budget_exceeded", message: "Your organization's AI budget ($<limit>/mo) has been reached. Contact your administrator to increase the limit or wait until next month.", spent_usd: err.spentUsd, limit_usd: err.limitUsd }`. This is the consistent mapper every wrapped route will use.

Use `createAdminClient` from `@/lib/supabase/server` for ALL ledger reads/writes (server/background per CLAUDE.md). Type the entitlements read row explicitly (`{ monthly_ai_budget_usd: number | null } | null`) and the sessions rows explicitly (`{ cost_usd: number | string | null }[]`). No `any`.

After Task 3 you may keep the cast or tighten it ŌĆö keep the build green either way.
