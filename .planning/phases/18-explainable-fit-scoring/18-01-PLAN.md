<objective>
Create the persistence layer for cited vault evidence: a new `rfp_fit_evidence` table (RLS-protected, ON DELETE CASCADE) plus a server-side store helper that writes cited chunks per (opp, org, dimension) and prunes superseded rows on rescore.

Purpose: SCORE-02 requires the score explanation to cite specific vault artifacts by name with an excerpt. Storing those citations in a dedicated table keeps `rfp_opp_matches.score_breakdown` compact for fast feed queries while giving the UI a queryable source of evidence. This plan is the data foundation other plans build on.
Output: One migration, one store module.
</objective>
